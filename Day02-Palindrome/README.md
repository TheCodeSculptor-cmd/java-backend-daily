# Day 02 - Palindrome

## Problem
Check whether a given string is palindrome.

## Approach
Use Two Pointers.

- One pointer starts from beginning.
- Another starts from the end.
- Compare characters.
- If all characters match, the string is a palindrome.

---

## Time Complexity
O(n)

## Space Complexity
O(1)

---

## Sample Input
level

## Sample Output
level is a Palindrome.
