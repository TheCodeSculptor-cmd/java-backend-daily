# Day 03 - Anagram Checker

## Problem
Check whether two strings are anagrams.

## Approach
1. Compare lengths.
2. Convert both strings to character arrays.
3. Sort both arrays.
4. Compare using Arrays.equal().

## Time Complexity
O(n log n)

## Space Complexity
O(n)

## Sample Input
listen
silent

## Sample Ouput
Anagrams

## Concepts learned
- String
- Character Arrays
- Arrays.sort()
- Arrays.equals()
