# Day 04 - Character Frequency

## Problem

Count the frequency of every character in a string.

---

## Approach

- Traverse the string.
- Store each character in a HashMap.
- If the character already exists, increment its count.
- otherwise, add it with count 1.

---

## Space Complexity
O(n)

---
## Sample Input
programming

## Sample Output

p -> 1
r -> 2
o -> 1
g -> 2
a -> 1
m -> 2
i -> 1
n -> 1
